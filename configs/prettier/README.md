# `@nzz/et-utils-config-prettier`

shareable prettier config

## Usage

```bash
npm install @nzz/et-utils-config-prettier -D
```

Edit `package.json`:

```jsonc
{
  // ...
  "prettier": "@nzz/et-utils-config-prettier"
}
```

## Official Docs

https://prettier.io/docs/en/configuration.html#sharing-configurations
