# ed-tech-utilities

Contains all utility packages managed by editorial-tech

## Development

1. `cd <package-folder>`
2. `npm run dev`

## Testing

1. `cd <package-folder>`
2. `npm run test`

## Publication Process

1. `cd <package-folder>`
2. `npm publish` (automatically runs tests, typescript compiler & rollup bundler)

## Install Package

`npm install <package-name>`
